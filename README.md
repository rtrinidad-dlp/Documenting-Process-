# Documenting-Process

## Tools Installed
- Cursor IDE
- Claude Code for VS Code (installed via Cursor's Editor Window)
- Codex - OpenAI's coding agent
- Git

## Steps Completed

### 1. Installing Cursor
Downloaded and installed the Cursor desktop application. 
Linked it to my GitHub account during setup.

### 2. Finding the Extensions
When I first opened Cursor, I couldn't find Claude Code or Codex 
in the plugins section. I searched YouTube for tutorials but the 
videos showed an "Extensions" tab that wasn't visible in my 
interface. After exploring the menu, I found that going to 
File → Open Editor Window opened a new interface similar to 
Visual Studio Code, where the Extensions tab was available.

### 3. Installing Claude Code
Searched for "Claude Code" in the Extensions tab and found it 
listed as "Claude Code for VS Code." Installed it, accepted the 
prompt that appeared, and it installed correctly. Clicked on the 
Claude Code icon and the chat opened immediately.

### 4. Installing Codex
Searched for "Codex" in the Extensions tab and installed 
"Codex - OpenAI's coding agent." It asked me to log in with 
my ChatGPT account, which I did successfully.

### 5. Creating the GitHub Repository
Went to my GitHub account and created a new public repository 
called "Documenting-Process-" with the README option enabled.

### 6. Cloning the Repository in Cursor
At first I couldn't find a way to open the repository from Cursor. 
After researching, I found that Git needed to be installed 
separately for Cursor to connect to GitHub. Once Git was installed, 
I clicked "Clone repo" and Cursor asked me to authenticate with 
GitHub using a one-time code. I copied the code, authorized 
Visual Studio Code, and successfully linked my account. I then 
searched for my repository using Ctrl+Alt and selected it from 
the list. Cursor asked me to choose a destination folder — I 
created a new folder on my desktop and selected it as the 
repository destination. The repository opened successfully.

### 7. Editing the README
Once the repository was open in Cursor, the README.md file was 
visible in the file explorer. I opened it and documented the 
entire process in this file.

## Problems I Ran Into and How I Solved Them

- **Couldn't find Extensions tab:** The default Cursor interface 
only showed plugins. I discovered that File → Open Editor Window 
revealed the full Extensions marketplace.

- **Couldn't find Claude Code and Codex after installing them:** 
After installation, I wasn't sure where to access them. I 
discovered they were accessible by clicking the dropdown arrow 
next to the four small icons at the top of the Extensions panel.

- **Clone repo button wasn't working:** Researched and found that 
Git needed to be installed separately. After installing Git and 
restarting Cursor, the clone function worked correctly.

- **GitHub authentication:** Cursor required a one-time code to 
authenticate with GitHub. I followed the prompts, authorized the 
connection, and it worked successfully.
